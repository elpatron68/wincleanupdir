# wincleanupdir
Deletes files from directory and subdir and keeps N newest files.

## Syntax:
`Cleanup.exe <Directory> <Search pattern> <Number of newest files to keep>`


## Example
`Cleanup.exe *.tmp "c:\path to my data" 5`

(Deletes all but the 5 newest files which matches the pattern `*.tmp` in `c:\path to my data` and each subdirectory)
